# 涨了么 🎾

> 上海网球 / 匹克球 / 板式网球找场地的「一个入口」——聚合全平台场馆信息，筛选完直接跳到对应小程序订场。

**👉 立即体验：<https://469910093-ui.github.io/zhangleme/>**（手机 / 微信内打开体验最佳）

---

## 解决什么问题

在上海想订一片网球场，你可能要在**韵动吧、久事体育、趣运动、勾勾运动、网球有约……十几个小程序之间来回切换**：

- 信息分散：每个平台只有自家场馆，没有全城视角
- 筛选太弱：想找「浦东 + 室内 + 有灯光 + 100 元以内」基本靠翻
- 比价困难：同一片区不同场馆价格差一倍，没法横向对比

**涨了么把这些信息聚合到一张列表 / 一张地图上**：先在这里筛选、比价、看场馆详情，选好之后一键跳转到对应平台的小程序完成预约。我们不做交易，只做「找场地」这一步。

## 怎么用

1. 打开 [体验页面](https://469910093-ui.github.io/zhangleme/)
2. 顶部选择运动类型（网球 / 匹克球 / 板式网球）
3. 用筛选条过滤：区域、室内外、价格区间、灯光 / 淋浴 / 停车等设施
4. 列表视图逐个看，或切换 **地图视图** 看场馆分布
5. 点进场馆卡片查看详情，点 **去预约** 跳转到该场馆所在平台（微信内可直接拉起小程序）
6. 顺手点个 ❤️ 收藏，下次直接在「我的收藏」里找

目前收录 **110+ 家上海场馆**，覆盖久事体育、韵动吧、趣运动、勾勾运动、久博、网球有约、多多动、帮帮体育、PadelU 等 **10+ 个预约渠道**（含商家自营和微信群预约）。

## 项目结构

| 文件 / 目录 | 说明 |
|---|---|
| `index.html` | 入口页，重定向到主应用 |
| `demo.html` | **主应用**：场馆列表、筛选、地图、收藏、预约跳转（单文件，无构建） |
| `admin.html` | 运营后台：场馆数据维护 |
| `demo_user_frontend.html` | 用户端早期原型（存档） |
| `api/wxsign.js` | 微信 JS-SDK 签名接口（Vercel Serverless，用于微信内跳转小程序） |
| `docs/` | 过程文档：商业分析、产品方案、技术规格（见下） |

## 技术说明

- 纯静态单页应用，**无框架、无构建步骤**，GitHub Pages 直接托管
- 地图使用 [Leaflet](https://leafletjs.com/)
- 微信内通过 JS-SDK `wx-open-launch-weapp` 拉起目标预约小程序；微信外提供复制小程序名等降级方案
- 场馆数据目前内置在页面中，人工维护（不做爬虫，原因见 [商业分析文档](docs/商业分析结论_强了么.md)）

本地预览：clone 后直接用浏览器打开 `demo.html` 即可。
